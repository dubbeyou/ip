# Snek Project

Welcome! This is a Java project with a Graphical User Interface (GUI) that allows the user to chat with Snek.

## Getting Started

### Prerequisites

- JDK 17

### Instructions

1. Download the latest JAR file [here](https://github.com/dubbeyou/ip/releases).
2. Run the application:
   - Using a command prompt or terminal, navigate to the directory containing the JAR file.
   - Run `java -jar snek.jar`
   - Enjoy!

## Notes
- Ensure that you are running JDK 17:
   - Run `java --version` in a command prompt or terminal.
